BismarkFiles#

class BismarkFiles.BismarkFiles(files, genome, flank_windows=500, gene_windows=2000, batch_size=1000000, cpu=4, line_plot=True, heat_map=False, bar_plot=False, box_plot=False, store_res=False)#

Class to process and plot multiple files

Parameters:

  • files (list[str]) – List with paths to bismark genomeWide reports

  • genome (DataFrame) – polars.Dataframe with gene ranges

  • flank_windows (int) – Number of windows for flank regions

  • gene_windows (int) – Number of windows for gene regions

  • batch_size (int) – Number of rows to read by one CPU core

  • cpu (int) – How many cores to use. Uses every physical core by default

  • line_plot (bool) – Whether to plot Line plot or not

  • heat_map (bool) – Whether to plot Heat map or not

  • bar_plot (bool) – Whether to plot Bar plot or not

  • box_plot (bool) – Whether to plot Box plot or not
